# Dummy Portfolio

Dummy portfolio created as a part of ChaiCode WebDev Cohort 2026

## Instruction for Setup & Uses

1. Download the code from GitHub repo, unzip.
2. Open the index.html file in your browser.
3. Alternately you can open the folder in VS Code, install "Live Server" extension and start the local server.

Feel free edit the files, change the dummy data with your own and deploy the code at your server.
